## Développement

Compilation:
```
sbt compile test:compile
```

Lancement des tests:
```
sbt test
```

Lancement de l'application:
```
sbt run
```

Packaging:
```
sbt package
```

Note: ajouter "~" devant une commande pour la lancer en mode "watch"
